# FIAP Bank ATM

Checkpoint 1 - Java

Sistema simples de caixa eletronico feito em Java puro, usando apenas Scanner para entrada de dados. Tudo roda no terminal.

## O que faz

- Cadastro do nome do cliente
- Criacao de senha forte (com validacao)
- Login com no maximo 3 tentativas
- Menu com opcoes de consultar saldo, depositar, sacar e sair

## Como rodar

```
javac FiapBankAtm.java
java FiapBankAtm
```
